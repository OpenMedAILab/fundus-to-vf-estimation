## 二、新增方案:预测→伪彩视野图,与真实视野做结构对比

### 2.1 方法(对首版全局-SSIM 零结果的三处修复)

首版 `external_ssim.py` 用**单一全局窗口** SSIM,得 matched −0.033 vs mismatched −0.037、p=0.47(完全无区分)。本版 `external_pseudocolor.py` 修复:

1. **页面筛选**:每次随访有 30°/50° **多页**(且无眼别标签)。50° 外周页中央留白、还有全黑退化页,与模型的"中央 ~30°"视野不可比 → 按"中央 30% 半径内有数据占比最高 + dB 结构(std)最大"自动**只选 30° 中央页**。
2. **OD/OS 朝向**:无眼别标签 → SSIM 取 {原图, 水平镜像} 的**较优**,消除左右眼镜像约定的干扰。
3. **局部窗口 SSIM**:改用 skimage 高斯加权局部 SSIM(data_range=33),**仅在圆盘内**平均;不再用只测全局均值/方差的单窗口。
4. 另加**上下半野(sup/inf)dB 差**的 matched 相关(对 OD/OS 镜像不变,临床上半野非对称是青光眼关键征)。
5. VF 访次级**去重**(每次随访只算一次),配对仍按"姓名+最近日期 ≤90 天"。

### 2.2 结果(n=22 配对访次,日期差中位 1 天)

| 指标 | 值 | 解读 |
|---|---|---|
| **结构 SSIM(matched)** | 0.483 ± 0.259 | |
| **结构 SSIM(mismatched)** | 0.459 ± 0.243 | |
| Mann-Whitney p(matched>mismatched) | **0.34 → 不显著** | ❌ 空间结构**无法区分** |
| 上下半野不对称 Spearman | r=0.11, **p=0.62** | ❌ 不能判别哪个半野更差 |
| **空间 MS(整体敏感度)Spearman** | **r=0.505, p=0.016** | ✅ 整体严重度**显著迁移** |

### 2.3 结论:整体严重度迁移,空间结构不迁移

- **标量信号(整体平均敏感度/MS)确实迁移**(伪彩曲面的盘内均值 Spearman 0.51,与 MS 代理法 0.48 一致、互相印证)。
- **空间缺损结构(伪彩图的形状/缺损位置)不迁移**:即使做了页面筛选 + OD/OS 镜像 + 局部窗口 SSIM,matched 与 mismatched 仍无差别(p=0.34),半野非对称也测不到(p=0.62)。

**为什么?** 见 `fig_external_pseudocolor.png`:
- 真实视野**接近正常(整片高 dB)** 的眼,SSIM 高(0.66–0.87)——但这只是"两边都亮",由整体水平驱动,不是结构吻合;
- 真实视野有**真实局灶缺损(蓝色暗区,诊断上最关键)** 的眼,模型的预测仍是一张**偏通用、中央高**的图,完全对不上 → SSIM 仅 0.04–0.49。
- `fig_pred_pseudocolor_gallery.png` 也显示:不同外部 CFP 的**预测伪彩图彼此高度相似**(预测范围压缩),自然无法匹配多样的真实缺损形态。

这与内部复现的"预测范围压缩、MS-R²~0.5 但逐点结构弱"完全自洽:**单张眼底图能估计视野的整体严重度(可排序),但跨设备/跨网格复原局灶缺损的空间拓扑超出其能力**。这是一个诚实、可写进修稿的阴性结果——支持把主张定位为"视野**功能(整体)**估计 + 临床决策支持",而非"替代视野检查/复原缺损图"。

## 三、数据账目与局限(诚实)

- 343 CFP 全部推理;119 VF 图按访次聚合得 38 访次,自动选出 38 张 30° 中央页;与 CFP 同次配对得 **22 个唯一访次**用于结构对比(比 MS 代理的 69 张少,因这里按访次去重且只用 30° 中央页)。
- SSIM 阴性是**真阴性而非样本量不足**:matched 与 mismatched 效应量极小(0.483 vs 0.459),增大 n 也不会显著。
- 仍无眼别标签 → 同一患者 OD/OS 真实缺损可能不同,选错眼会加噪(已用镜像 + 半野不变量缓解,但无法完全消除)。
- image→dB 仍是灰度代理(~12 档 + 插值),空间结构对比受其分辨率限制。
- **解锁项**:若外部提供每眼数值 VF(逐点 dB)或眼别标注,即可把结构对比从"灰度代理"升级为论文级逐点 MAE/结构一致性。
