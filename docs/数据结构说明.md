## 三、ckpt/ 权重命名规范（116 个）

命名统一为：**`{task}_{arch}_{input}[_变体]_s{seed}/best.pth`**（每个目录含 `best.pth` + `result.json`）。

| 字段 | 取值 | 含义 |
|---|---|---|
| **task** | `reg`(50) / `cls`(66) | reg=视野 59 点回归(VF estimation)；cls=PLR3 进展二分类 |
| **arch** | `resnet` / `transformer` / `hybrid` / `full_hybrid` | 单模型 / ViT / 拼接 / 交叉注意力(论文最终模型) |
| **input** | `cfp` / `roi` / `annotated` | 全眼底 / ROI / ROI+OD-OC 标注 |
| **变体** | `cfp_aug` | 输入 cfp **+ StyleGAN 合成图增强** |
| （cls 专有） | `ce_nosamp` / `ce_samp` / `focal_samp` | R4-5 逐步增量消融(纯CE→+采样→+Focal) |
| （reg 专有） | `cfp_conv` | 收敛曲线专用(单 seed) |
| **seed** | `s0 / s1 / s2` | 3 个随机种子(多数配置) |

**关键权重（最常用，推理/出图直接加载）：**
- `reg_hybrid_cfp_s0/1/2` — **回归最优**，外部验证 + 临床误差分析 + 路线B 都用它
- `reg_full_hybrid_cfp_s0/1/2` — 论文最终模型(交叉注意力)
- `cls_full_hybrid_cfp_s0/1/2` — **分类最优**(ROC-AUC 0.728)
