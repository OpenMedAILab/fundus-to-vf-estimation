# 皖南外部验证 — 预测→伪彩→相似度（独立重做）

> 任务：用训练好的 Hybrid 模型对外部眼底照（CFP）预测 59 点 VF 数值 → 渲染伪彩视野曲面 →
> 与真实 VF「Level」灰度图（凡例较准为 dB 曲面）做相似度对比，完成外部验证。
> **独立新实现**（从生数据与模型出发重写，未复用既有外部脚本）。全程 inference-only、无微调、无合成图。
> 环境：服务器 10.11.154.191，conda env `ffs`（torch 2.4.1 / skimage 0.26 / cv2 4.11），GPU2。
> 脚本：`reproduce_scripts/wannan_ext_val.py`　结果：`wannan_extval.json`、`wannan_extval_pairs.json`、4 张图。

---

## 一、数据与方法（关键改进点）

**外部数据（皖南）**：CFP 343 张（130 个「姓名+日期」访次）；VF「Level」灰度图 119 张（38 访次）。
仅图像、**无数值 VF、无眼别标签**，设备为 Ooto AP-100（与训练集 GRAPE 不同中心）。

针对这种「只有图」的外部集，本次做了 4 处第一性原理设计（均从样本实测标定）：

1. **凡例 LUT 较准**：每张 Level 图左侧自带灰阶色标 `X,0,3,…,30,>30`。实测灰度→dB 为**非线性**：
   `gray {0,96,117,141,172,193,210,220,228,236,242,248,254} → dB {0,0,3,6,9,12,15,18,21,24,27,30,33}`。
   **黑(0)=「X」=绝对暗点≈0 dB**（不是"无数据"）；白(255)=格线/标注。→ 比简单线性 `g/255·33` 更忠实。
2. **几何标定**（934×934）：圆心=图像中心；30°边界=0.41·W（恰穿过"30"刻度框）；**单位 27°=0.37·W**
   （对齐模型 59 点布局的归一化尺度，已用 59 点叠加图核验对齐，见 `wannan_fig_debug_extract.png`）。
3. **格线/标注掩膜**：白(≥253)膨胀 4px 吞掉刻度框内黑字 → 圆盘内最近邻插值修补，避免黑字被误读为暗点。
4. **页面筛选**：每访次多页 = {30°中央, 50°外周} ×(可能左右眼)。50°页中央留白(白)→中央有数据占比低；
   全黑/退化页→结构 std≈0。规则：**中央有数据占比>0.5 且 结构 std>1.5，取结构最丰富的 30°中央页**。

**配对**：姓名 + 最近日期 ≤90 天（中位日期差 1 天）。CFP 同访次多图取预测均值。
**对比**：预测 59 点（dB）与真实 dB 曲面，统一渲染到 27° 正准圆盘。OD/OS 未知 → 结构比较取 {原图, 水平镜像} 较优。

最终：**n = 21** 个干净配对访次（37 个有效中央页里，21 个有 ≤90 天的同名 CFP 访次；瓶颈是 CFP/VF 不总同次）。

## 二、结果（n=21）

| 维度 | 指标 | 值 | 结论 |
|---|---|---|---|
| **空间结构** | SSIM matched vs mismatched | 0.414 vs 0.413，Mann-Whitney **p=0.44** | ❌ 结构不可区分 |
| **半野定位** | 上下半野(sup−inf)dB Spearman | r=−0.03，**p=0.89** | ❌ 不能判别哪半野更差 |
| **逐点结构** | 每眼 pred59 vs real59 Spearman | 中位 **0.12**（均 0.10） | ❌ 近乎无逐点相关 |
| **整体严重度** | 预测 MS vs 真实 MS（59测点均）Spearman | **r=0.34，p=0.14**；Pearson R²=0.13 | △ 弱正相关，**未达显著** |
| （对照：盘面均 MS） | Spearman | r=0.38，p=0.09 | △ 同上 |
| | MS-MAE / 校准后 | 6.4 / 5.8 dB | 尺度系统偏差大 |

## 三、解读

1. **空间缺损结构不迁移（稳健）**：即便用了凡例较准 + 几何对齐 + 格线掩膜 + 镜像不变 + 局部窗口 SSIM，
   matched 与 mismatched 仍无差别（p=0.44）；半野非对称、逐点相关也都接近 0。
   `wannan_fig_debug_extract.png` 直观显示：真实野能精确提取出**局灶暗点**（如颞侧蓝色缺损），
   但**预测伪彩图是一张偏通用、中央偏高的图**，对不上具体缺损位置。
   `wannan_fig_pred_gallery.png` 进一步显示不同 CFP 的预测伪彩图**彼此高度相似（范围压缩）**。

2. **整体严重度仅弱正相关、未显著**：`wannan_fig_ms_scatter.png` 可见正趋势，但被**重度端塌缩**拖累——
   真实重度野（MS 0–5，灰度图整片暗，已正确提取）模型仍预测到 **MS≈14–24**（无法预测低敏感度）。
   模型把"近正常 vs 中度"排得还行，"中度 vs 重度"几乎排不开。这与内部复现"预测范围压缩、逐点结构弱"自洽。
   在干净访次级 n=21 下，r≈0.34（p≈0.14）是**真实但微弱**的关联：受限于①样本量（仅 21 对）②重度端塌缩
   ③无眼别标签（同一患者 OD/OS 可能选错眼，加噪）。

3. **对论文的含义**：这条"预测→伪彩→SSIM"路线**不支持**"复原缺损图 / 替代视野检查 / 逐点结构验证"的主张；
   它最多支持"**视野整体功能（严重度）估计**"，且在本外部集该证据也偏弱。
   **建议**：外部严重度主张优先用直接的 **预测MS↔真实MS** 数值证据（而非伪彩 SSIM）来陈述，
   并把伪彩图仅作**定性可视化/失败案例**展示（诚实呈现"严重度可估、缺损形态不可复原"）。

---

## 四、局限与可加强项

- **n=21、无眼别、灰度代理**：真实 VF 只有图像、无逐点数值，dB 经 ~12 档凡例反推（分辨率有限）。
- **解锁项**：若外部能提供①**逐点数值 dB** 或②**眼别标注**，可把比较从灰度代理升级为论文级逐点 MAE，
  并做眼别解析配对（预计能让严重度相关更干净，但**不会改变结构不迁移的结论**）。
- 若需更大 n 的严重度证据，可在眼别解析后放宽到全部有效 VF 页配对（以更多配对眼提升统计功效）。
